# Config para ChatGPT

ChatGPT no usa un archivo de configuración local como Claude. La conexión se crea desde el producto.

Rellena estos campos:

- Server URL: `https://rpeak.vercel.app/api/mcp`
- Server label: `rpeak`
- Description: `Herramientas de entrenamiento de RPeak`
- Auth: `Bearer token`
- Token: `MCP_BEARER_TOKEN`

Pasos:

1. Activa `Developer mode` en ChatGPT.
2. Ve a `Settings -> Apps -> Advanced settings -> Create app`.
3. Crea la app con la URL anterior.
4. Pega el bearer token cuando te lo pida.
